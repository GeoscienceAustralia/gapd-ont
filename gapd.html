Abstract

This ontology represents many of the classes of information object that Geoscience Australia (GA) delivers publicly and the relationships between them.

Software Repositoryc back to ToC or Class ToC

IRI: http://pid.geoscience.org.au/def/ont/gapd#SoftwareRepository

A Software Repository is a system such as GitHub or BitBucket that stores and manages software code. A Software Repository is also a type of dataset therefor GA's Software Repositories are catalogued in one of GA's catalogues, just as all datasets are.

Web Servicec back to ToC or Class ToC

IRI: http://pid.geoscience.org.au/def/ont/gapd#WebService

GA's Web Services are a specialised type of a dataset distribution: they are a services that allow for dataset subsetting (querying) and supersetting (merging data from multiple datasets). GA has an extensive catalogue of Open Geospatial Consortium web services but also has a few non-OGC web services, such as bespoke RESTful APIs, database XML APIs and SPARQL endpoints.
