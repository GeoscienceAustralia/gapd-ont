Abstract

This ontology represents many of the classes of information object that Geoscience Australia (GA) delivers publicly and the relationships between them.

Public Datasetc back to ToC or Class ToC

IRI: http://pid.geoscience.org.au/def/ont/gapd#PublicDataset

A Public Dataset is a dataset that GA publishes publicly. It is not necissarily freely available or even avialable at all but its existence, i.e. its most basic metadata, and likely its complete metadata is freely available. GA' Public Datasets carry certain guarentees: their metadata will be accessible in certain formats (if it is a dataset in the main GA catalogue, then in ISO 19115-3-compliant metadata) and they will have certain values present, such as a custodian, abstract, license etc. These values are DCAT properties that this ontology mandates.

Public Ontologyc back to ToC or Class ToC

IRI: http://pid.geoscience.org.au/def/ont/gapd#PublicOntology

A Public Ontology is an ontology, which is a specialised kind of dataset, that GA publishes publicly. It is not necissarily freely available or even avialable at all but its existence, i.e. its most basic metadata, and likely its complete metadata, and probably all of its data is freely available. GA' Public Ontologies carry certain guarentees: their metadata will be accessible in certain formats (all Public Ontologies will have DCAT metadata and some OWL ontology metadata) and they will have certain values present, such as a custodian, abstract, license etc. These values are DCAT properties that this ontology mandates.

Public Web Servicec back to ToC or Class ToC

IRI: http://pid.geoscience.org.au/def/ont/gapd#PublicWebService

A GA Public Web Service is a dataset that GA publishes publicly. It is not necissarily freely available or even available at all perhaps without some granted level of access but its existence, i.e. its most basic metadata, and likely its complete metadata is freely available. GA Public Web Services carry certain guarentees: their metadata will be accessible in certain formats (always ISO 19115-3 compliant metadata) and will have certain values present, such as a custodian, abstract, lineage (the datasets the web service draws from) etc.

Web Servicec back to ToC or Class ToC

IRI: http://pid.geoscience.org.au/def/ont/gapd#WebService

GA's Web Services are a specialised type of a dataset distribution: they are a services that allow for dataset subsetting (querying) and supersetting (merging data from multiple datasets). GA has an extensive catalogue of Open Geospatial Consortium web services but also has a few non-OGC web services, such as bespoke RESTful APIs, database XML APIs and SPARQL endpoints.
